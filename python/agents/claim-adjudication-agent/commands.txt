1. Environment & Dependencies
Use these to set up or update your Python environment using uv.

bash
# Initial sync of all dependencies
uv sync
# Sync development dependencies (required for testing/evaluation)
uv sync --dev
# Update lockfile to latest versions
uv lock --upgrade
2. Google Cloud Authentication
Run these if you encounter any "RefreshError" or "Unauthorized" issues with the Vertex AI or GCS tools.

bash
# Provide Application Default Credentials (ADC)
gcloud auth application-default login
# Set the quota project for API billing
gcloud auth application-default set-quota-project proj-genai
3. Running the Agent
You must run these through uv run to ensure the agent uses the local virtual environment.

bash
# Start the Web Interface (Localhost UI)
uv run adk web
# Run the agent via CLI for quick testing
uv run adk run claim_adjudication_agent
4. Testing & Evaluation
Use these to verify the agent's logic and performance.

bash
# Run all tests (Functional + Evaluation)
uv run pytest
# Run functional agent tests only
uv run pytest claim_adjudication_agent/tests/test_agents.py
# Run the automated evaluation suite only
uv run pytest claim_adjudication_agent/eval/test_eval.py
